## 🏗️ Tech Stack

### **Frontend**
- **Framework:** React 18.2
- **Build Tool:** Vite 5.1
- **Routing:** React Router 6.22
- **Animations:** Framer Motion 11.0
- **Styling:** CSS Modules
- **Icons:** React Icons 5.0

### **Premium Features**
- **Particles:** tsParticles 3.0
- **Code Highlighting:** React Syntax Highlighter
- **Carousel:** Swiper 11.0
- **Counting:** React CountUp 6.5
- **Typing:** React Type Animation 3.2
- **Forms:** EmailJS 3.2

### **Performance**
- **PWA:** Vite Plugin PWA
- **Compression:** Gzip & Brotli
- **Analytics:** Bundle Visualizer
- **Caching:** IndexedDB + Memory Cache

## 📦 Installation

### Prerequisites
- Node.js 16+ and npm
- Git

### Quick Start

```bash
---# 1. Clone the repository
git clone https://github.com/Dev-moe-kyawaung/Premium-advanced-Portfolio.git
cd Premium-advanced-Portfolio

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env
# Edit .env with your credentials

# 4. Start development server
npm run dev

# 5. Open browser
# Visit http://localhost:3000
